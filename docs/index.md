# Moderator Toolbox-NXG

Moderator Toolbox-NXG is a browser extension that provides a set of tools to help Reddit moderators do their jobs more effectively. It is an extended fork of [Toolbox for Reddit](https://github.com/toolbox-team/reddit-moderator-toolbox), building on the original with improvements to the wiki storage schema, a new usernotes layout with archiving support, a richer config editor, and ongoing refactors for maintainability.
