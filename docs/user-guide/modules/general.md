# General Settings

**Platforms:** Old Reddit and New Reddit · **Always on**

General Settings holds cross-module preferences that affect the overall Toolbox-NXG experience.

## Overview

These settings apply globally across all modules. They cover notification delivery style and the behaviour of the context menu that appears when Toolbox shows new activity.

## Settings

| Setting                       | Default | Description                                                                       |
| ----------------------------- | ------- | --------------------------------------------------------------------------------- |
| Native notifications          | On      | Show OS-level notifications instead of in-page banners                            |
| Context menu location         | Left    | Which side of the screen the context menu slides in from                          |
| Context menu attention effect | Open    | How the context menu signals new items: open automatically, fade in, or no effect |
| Context menu click to open    | Off     | Require a click to open the context menu instead of opening on hover              |
