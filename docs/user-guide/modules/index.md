# Modules

Toolbox-NXG functionality is organized into modules. Click any module to see its documentation.

## Always-on modules

These modules cannot be disabled.

- [General Settings](general.md) — cross-module preferences
- [Modbar](modbar.md) — persistent bottom toolbar
