# Queue Overlay

**Platforms:** Old Reddit and New Reddit · **Default:** Enabled

Queue Overlay opens mod queues in an in-page overlay, letting you moderate without navigating away from your current page.

## Overview

When you click a queue link in the Modbar (or, on old Reddit, when configured to do so), the queue opens in a floating overlay panel instead of loading a new page. You can moderate items in the overlay and dismiss it to return to whatever you were reading.

## Features

**Queue in overlay from Modbar (new Reddit)** — clicking the queue or unmoderated icons in the Modbar on new Reddit opens the corresponding old Reddit queue in an overlay, giving you the familiar old Reddit queue experience while staying on the new Reddit page.

**Queue in overlay from Modbar (old Reddit)** — optionally, the same behavior applies when clicking queue links in the Modbar on old Reddit.

## Settings

| Setting                                        | Default | Description                                                                     |
| ---------------------------------------------- | ------- | ------------------------------------------------------------------------------- |
| Open queue in overlay from Modbar (new Reddit) | On      | Open old Reddit queue in overlay when clicking Modbar queue icons on new Reddit |
| Open queue in overlay from Modbar (old Reddit) | Off     | Open queue in overlay when clicking Modbar queue icons on old Reddit            |
