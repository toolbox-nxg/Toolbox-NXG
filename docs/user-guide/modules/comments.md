# Comments

**Platforms:** Old Reddit and New Reddit · **Default:** Enabled

The Comments module adds comment management tools and viewing enhancements to comment threads and moderation pages.

## Overview

Comments improves the experience of reading and moderating comment threads. Most enhancements work on both platforms; some are old Reddit only.

## Features

**Context popup link** — adds a link where appropriate to open the full comment context in an in-page popup, without leaving the current page.

**Keyword highlighting** — highlights any keywords you specify anywhere in comment text. Configure one keyword per line in the module settings.

**Hide removed comments** _(old Reddit)_ — hides removed comments by default. They can be revealed individually.

**Approve button on all comments** _(old Reddit)_ — shows an approve button on every comment, not just flagged ones.

**Spam button on ham-removed comments** _(old Reddit)_ — shows a "spam" button on comments removed as ham, letting you re-remove them as spam.

**Remove (not spam) on spam-removed comments** _(old Reddit)_ — shows a "remove" button on spam-removed comments, letting you convert the classification to a regular remove.

**Hide old comments button** _(old Reddit)_ — adds a button to collapse comments older than a threshold, helping focus on recent activity.

## Settings

| Setting                                    | Default   | Description                                            |
| ------------------------------------------ | --------- | ------------------------------------------------------ |
| Open context in popup                      | On        | Add a popup-context link to comments where appropriate |
| Highlight keywords                         | _(empty)_ | Keywords to highlight in comment text (one per line)   |
| Hide removed comments                      | Off       | Hide removed comments by default _(old Reddit)_        |
| Show approve button on all comments        | Off       | Approve button on every comment _(old Reddit)_         |
| Spam button on ham-removed comments        | Off       | Spam button on non-spam removals _(old Reddit)_        |
| Remove (not spam) on spam-removed comments | Off       | Remove button on spam removals _(old Reddit)_          |
| Show hide-old-comments button              | On        | Button to collapse old comments _(old Reddit)_         |
