# Comment Triage

**Platforms:** Old Reddit only · **Default:** Disabled

Comment Triage helps moderators prioritize and process problem comments by surfacing low-scoring and controversial comments.

## Overview

Comment Triage operates on old Reddit comment pages. It can automatically highlight comments with low scores or high controversy, and surface them visually so they don't get missed in long threads.

## Features

**Auto-highlight** — automatically applies a highlight colour to downvoted or controversial comments without requiring manual review.

**Negative score threshold** — sets the score at which a comment is highlighted as problematic.

**Controversial comment highlighting** — highlights comments that are highly controversial (high upvote/downvote ratio).

**Expand on load** — automatically expands all downvoted and controversial comments when the page loads, even if they were collapsed by Reddit.

**Sort on "load more"** — continues the sort-and-highlight behaviour when additional comments are loaded via "load more comments" links.

**Comment child count** — always shows the number of child replies a comment has, even when the comment is collapsed.

**Child count in upper-left** _(advanced)_ — shows the child count in the upper-left corner of each comment. May slightly affect page layout.

## Settings

| Setting                              | Default | Description                                              |
| ------------------------------------ | ------- | -------------------------------------------------------- |
| Highlight comments automatically     | Off     | Auto-highlight downvoted/controversial comments          |
| Negative comment highlight threshold | 0       | Score at or below which a comment is highlighted         |
| Highlight controversial comments     | On      | Highlight highly contested comments                      |
| Expand on load                       | Off     | Expand all downvoted/controversial comments on page load |
| Sort on "load more comments"         | Off     | Continue sorting when additional comments are loaded     |
| Always display number of children    | On      | Show child count on every comment                        |
| Display child count in upper-left    | Off     | Show child count in upper-left corner _(advanced)_       |
