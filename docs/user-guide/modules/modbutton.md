# Mod Button

**Platforms:** Old Reddit and New Reddit · **Default:** Enabled

Mod Button registers moderation action buttons next to usernames, providing quick access to ban, mute, and other per-user actions.

## Overview

The Mod Button appears next to usernames on moderation pages. Clicking it opens a dropdown of actions you can take against that user in the current subreddit. A global action button can optionally apply the same action across multiple subreddits at once.

## Features

**Per-user action dropdown** — ban, unban, mute, and other mod actions available from a single dropdown next to any username.

**Remember last action** — pre-selects the last action you used when you open the dropdown again.

**Saved subreddits** — pin subreddits to the dropdown for quick access to their queues and mod pages.

**Global Action button** _(advanced)_ — adds a button that applies the selected action across all moderated subreddits, with an exclusion list for subreddits you want to skip.

## Settings

| Setting                          | Default   | Description                                                         |
| -------------------------------- | --------- | ------------------------------------------------------------------- |
| Saved subreddits                 | _(empty)_ | Subreddits pinned to the Mod Button dropdown                        |
| Remember last action             | Off       | Pre-select the last-used action when opening the dropdown           |
| Enable Global Action button      | Off       | Add a button for cross-subreddit actions _(advanced)_               |
| Exclude subs from Global Actions | _(empty)_ | Subreddits to skip when using the Global Action button _(advanced)_ |
