# History Button

**Platforms:** Old Reddit and New Reddit · **Default:** Enabled

History Button adds per-user submission and comment history popups, accessible directly from a username link on any Reddit page.

## Overview

Clicking the history button next to a username opens an overlay showing that user's recent submissions and comment history. This gives moderators quick context about a user without navigating away from the current page.

## Features

**Submission history** — shows the user's recent submissions with vote counts, subreddits, and dates.

**Comment history** — shows the user's recent comments with context links. Comment history is loaded alongside submissions by default.

**NSFW filtering** — NSFW submissions can be excluded from history results.

**Configurable fetch depth** — controls how many comments are retrieved per user history lookup (100–1000).

## Settings

| Setting                          | Default | Description                                                              |
| -------------------------------- | ------- | ------------------------------------------------------------------------ |
| Load comment history immediately | On      | Load comment history at the same time as submission history _(advanced)_ |
| Number of comments to retrieve   | 1000    | How many comments to fetch per history lookup _(advanced)_               |
| Include NSFW submissions         | Off     | Include NSFW posts in history searches                                   |
