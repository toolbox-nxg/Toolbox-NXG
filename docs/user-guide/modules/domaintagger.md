# Domain Tagger

**Platforms:** Old Reddit and New Reddit · **Default:** Enabled

Domain Tagger colour-tags link domains on post listings, helping moderators quickly identify known-good or known-bad domains at a glance.

## Overview

Domain tags are configured per-subreddit in the Config overlay (Domain Tags tab). Each entry maps a domain name to a color. When Domain Tagger encounters a link post whose domain matches a configured entry, it renders a visual indicator in the style selected in your personal settings.

## Features

**Visual domain indicators** — shows a colored badge, dot, border, or background on link posts whose domain has been tagged. Five display styles are available.

**Per-subreddit tag lists** — tags are stored in the subreddit config and shared with all moderators of that subreddit.

## Settings

| Setting      | Default   | Description                                                                                                  |
| ------------ | --------- | ------------------------------------------------------------------------------------------------------------ |
| Tag location | Title dot | Where to show the domain tag — options: title dot, post border, post title, domain background, domain border |
