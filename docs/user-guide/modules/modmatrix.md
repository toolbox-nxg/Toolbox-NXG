# Mod Log Matrix

**Platforms:** Old Reddit and New Reddit · **Default:** Enabled

Mod Log Matrix adds a mod log matrix button to both old and new Reddit, providing a visual breakdown of moderator activity by action type.

## Overview

Clicking the Mod Log Matrix button on a subreddit opens an overlay that fetches the subreddit's mod log and displays a matrix view: moderators on one axis, action types on the other, with counts at each intersection. This makes it easy to see at a glance who has been doing what.

## Features

**Action matrix** — tabular breakdown of mod log entries by moderator and action type (approve, remove, ban, etc.).

**Mod log link** — each cell links to the filtered mod log for that moderator/action combination.

## Settings

Mod Log Matrix has no user-facing settings.
