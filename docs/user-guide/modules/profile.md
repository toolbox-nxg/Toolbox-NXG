# Profile Pro

**Platforms:** Old Reddit and New Reddit · **Default:** Enabled

Profile Pro provides an in-page user profile overlay with usernotes, mod log history, and other moderator-relevant information about a user.

## Overview

Clicking the profile button next to a username (or visiting a user profile page) opens the Profile Pro overlay. It aggregates usernotes, recent mod log entries involving the user, and submission/comment history into a single panel — giving moderators the context they need without navigating to multiple pages.

## Features

**Profile button next to usernames** — a small button next to any username opens the Profile Pro overlay for that user.

**Usernotes integration** — the overlay shows all usernotes for that user across every subreddit where you are a moderator.

**Mod log history** — recent moderator actions involving the user are listed inline.

**Submission and comment history** — recent activity is available from within the overlay.

**Legacy profile redirect** — optionally redirect profile page clicks to the old Reddit user profile (`old.reddit.com/user/…`).

**Subreddit color borders** — when Mod View Enhancements' subreddit color setting is enabled, the profile overlay can display the same colors.

## Settings

| Setting                                      | Default | Description                                              |
| -------------------------------------------- | ------- | -------------------------------------------------------- |
| Always open Toolbox profile on profile pages | Off     | Replace the Reddit profile page with the Toolbox overlay |
| Show profile button next to usernames        | On      | Show the profile button inline next to usernames         |
| Open profile links in legacy Reddit          | Off     | Redirect profile link clicks to old.reddit.com           |
