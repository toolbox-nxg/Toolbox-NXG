# Support Module

**Platforms:** Old Reddit and New Reddit · **Always on**

The Support module makes it easy to attach Toolbox-NXG debug information when you report a problem on the [/r/toolbox_nxg](https://www.reddit.com/r/toolbox_nxg/) subreddit. It is a small always-on helper with no user-facing settings.

## Overview

When you write a post on `/r/toolbox_nxg/submit` or a comment on a thread in `/r/toolbox_nxg`, the module adds an **Insert debug info** button to the text controls beside the text box. Clicking it appends a short, formatted table of diagnostic details to your text so the developers have what they need to investigate.

The button only appears on the `/r/toolbox_nxg` subreddit — it never adds anything to your posts or comments anywhere else.

## What's included in the debug info

The inserted table contains a handful of non-identifying technical details:

- Toolbox-NXG version
- Browser and platform information (your user-agent string)
- Whether debug mode is enabled
- Whether Modbar compact mode is enabled
- Whether advanced settings are enabled
- Whether cookies are enabled

You can review and edit the inserted text before submitting.

## Settings

This module has no user-facing settings.
